# Music Player (React Native App)

A modern **Music Player application built using React Native (Expo)**.
The app allows users to browse songs, artists, albums, search music, and play songs with a full player interface.

---

# 📱 Features

## Home Screen

* Suggested music
* Recently played songs
* Most played songs
* Artist list
* Album list
* Folder

## Songs Tab

* List of songs
* Sort songs (Ascending / Descending)
* Song row with artwork
* Song options menu (⋮)
* Song actions:

  * Play next
  * Add to queue
  * Add to playlist
  * Share
  * Delete

## Artists

* Artists list
* Artist options menu
* Artist details screen
* Play / Shuffle songs from artist

## Albums

* Album grid layout
* Album details screen
* Album statistics (songs, year, duration)
* Shuffle / Play buttons

## Search

* Search songs by keyword
* Recent search history
* Search results screen
* Empty search state

## Music Player

* Full player screen
* Album artwork
* Song title and artist
* Progress bar
* Seek functionality
* Controls:

  * Previous
  * Rewind 10 seconds
  * Play / Pause
  * Forward 10 seconds
  * Next
* Additional player options:

  * Playback speed
  * Sleep timer
  * Cast
  * More options

## Mini Player

* Floating mini player
* Quick play / pause
* Next song
* Tap to open full player

---

# 🧑‍💻 Tech Stack

* **React Native**
* **Expo**
* **React Navigation**
* **Expo AV (Audio Playback)**
* **Zustand (State Management)**
* **Saavn API (Music Search)**

# 🚀 Getting Started

## 1️⃣ Install dependencies

```
npm install
```
---

## 2️⃣ Start the development server

```
npx expo start
```

Run on:
* Physical device using Expo Go
